# DrD Radio

Интернет-радио с веб-интерфейсом для диджея и поддержкой ESP32-слушателей.

## 🚀 Быстрый старт

### Бэкенд (Flask)
```bash
cd backend
pip install -r requirements.txt
python app.py
```

# Фронтенд (React)
```bash
cd frontend
npm install
npm run dev
```
# 📡 Доступные страницы

Главная панель диджея: http://localhost:5173/

Техническая страница: http://localhost:5173/debug/radio

Прямой поток: http://localhost:5000/api/radio/stream

# 🎯 Возможности

Управление эфиром (выбор треков, плейлистов)

Поддержка микрофона

Подсчет слушателей

Потоковое вещание для ESP32

# 🚀 Возможности другого быстрого старта

### Открыть start_auto.bat 

Проект сам запустится и откроет Web страницу
